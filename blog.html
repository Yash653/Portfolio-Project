Box Model

Frontend, HTML CSS

The CSS box model is essentially a box that wraps around every HTML element. It consists of: margins, borders, padding, and the actual content.​ ​.

Flex Box

Frontend, HTML CSS

The flexbox is a layout module. It is used to make it easier to design flexible layout structure without using float and positioning.​

Gap Property

Frontend, HTML CSS

The Gap property defines the size of the gap between the rows and between the columns in flexbox.​

Properties​

Row-gap:-​It is used to set the gap between the rows.​

column-gap:-​It is used to set the gap between the columns.​

Functions in javascript
